### Here is a cheat sheet for common LaTeX equations: 

| Expression         | LaTeX Syntax              |
|---------------------|---------------------------|
| Fractions          | `\frac{a}{b}`             |
| Exponentiation     | `x^n`                     |
| Subscripts         | `x_n`                     |
| Summation          | `\sum_{i=1}^n x_i`        |
| Integrals          | `\int_a^b x dx`           |
| Matrices           | `\begin{bmatrix} a & b \\ c & d \end{bmatrix}` |
